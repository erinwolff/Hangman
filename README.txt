Download "Super Cool & Fun Hangman Game" folder
Open hangman.exe to begin the game. DO NOT DELETE ASSETS - THEY ARE NEEDED FOR GAME TO RUN.

HOW TO PLAY:
Make a guess - Use keyboard to input a letter (case-insensitive). 
Move onto the next guess - Press SPACE BAR or DIRECTIONAL KEYS (UP, DOWN, LEFT, RIGHT). 

The game is over after guessing incorrectly 6 times.

Good luck, have fun :) <3
